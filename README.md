# Traffic Signal Data Analysis

## Overview
This project analyzes traffic signal data to identify peak hours,
high-traffic intersections, and congestion patterns using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Evening hours show peak congestion.
- Signal S3 handles the highest traffic.
- Midday traffic is relatively lower.
